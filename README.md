# PLANEL

Веб-приложение для планирования: задачи, цели, привычки и аналитика в одном месте.

## О проекте

PLANEL — это лёгкий планировщик в формате SPA. Один HTML-файл, без сборки и зависимостей: открывается напрямую в браузере. Подходит как прототип для отработки UX до перехода на полноценный фреймворк.

### Ключевые разделы

- **Дашборд** — задачи на сегодня, помодоро-таймер, привычки дня
- **Задачи и доски** — Kanban с тремя статусами + фильтры
- **Планер** — календарь задач и событий
- **Цели и проекты** — стратегические инициативы с прогресс-барами
- **Привычки** — недельный журнал и статистика стабильности
- **Аналитика** — эффективность недели и загрузка по проектам
- **Настройки** — профиль, уведомления, команда, интеграции

## Технологии

- HTML5 + ванильный JavaScript (SPA-навигация без роутера)
- [Tailwind CSS](https://tailwindcss.com/) через CDN (плагины `forms`, `typography`)
- Google Fonts: `Inter` (UI), `Manrope` (заголовки)
- SVG-иконки inline, без внешних библиотек

## Дизайн-система

- **Палитра:** сине-голубая (`#1F76F2` — основной бренд, `#06B6D4` — поддерживающий cyan)
- **Тема:** только светлая
- **Контраст:** все тексты соответствуют WCAG AA
- **Сетка:** 8/12-колоночная, отступы кратны 4 px
- **Адаптив:** проверено на 380 / 452 / 768 / 1024 / 1440

## Структура

```
PLANEL/
├── src/
│   └── index.html        # всё приложение
├── docs/
│   └── screenshots/      # макеты для документации
└── README.md
```

## Запуск

Любым статическим сервером, например:

```bash
python3 -m http.server 8765
# затем открыть http://localhost:8765/src/index.html
```

Или просто открыть `src/index.html` двойным кликом.
